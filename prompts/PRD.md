# 프로필명 생성기 PRD

## 개요
한국어 단어 조합을 활용한 랜덤 프로필명 생성 웹 앱

## 기능 요구사항

### 1. 프로필명 생성 (메인 페이지 `/`)
- 글자수 선택: **8자 / 10자 / 12자**
- 1회에 1개의 프로필명 생성
- 생성된 프로필명 카드 형태로 표시
- **저장하기** 또는 **다시 생성** 선택 가능
- 저장 후 성공 피드백 표시

### 2. 한국어 단어 조합 규칙
- 형용사(adj), 부사(adv), 동사(verb), 명사(noun) 활용
- 선택된 글자수에 **정확히** 일치하는 프로필명 생성
- 템플릿 방식: 각 글자수별 조합 패턴 정의 후 랜덤 선택
- 동일 단어 중복 사용 방지

#### 글자수별 조합 예시
| 글자수 | 패턴 예시 | 생성 예시 |
|--------|-----------|-----------|
| 8자 | 형용사(4)+명사(2)+명사(2) | 아름다운별빛하늘 |
| 8자 | 형용사(3)+명사(2)+명사(3) | 따뜻한달빛무지개 |
| 10자 | 형용사(4)+명사(3)+명사(3) | 아름다운봄바람무지개 |
| 10자 | 부사(4)+형용사(2)+명사(2)+명사(2) | 살랑살랑맑은봄날하늘 |
| 12자 | 형용사(4)+형용사(4)+명사(2)+명사(2) | 아름다운반짝이는달빛하늘 |
| 12자 | 형용사(4)+동사(4)+명사(2)+명사(2) | 아름다운날아가는달빛하늘 |

### 3. 저장된 프로필명 페이지 (`/saved`)
- 저장된 모든 프로필명 목록 표시
- **정렬 기준** (내림차순):
  1. 글자수 (12자 → 10자 → 8자)
  2. 동일 글자수 내: 최신 저장순
- 각 항목: 프로필명, 글자수 뱃지, 저장 일시 표시
- 빈 상태(저장 없음) 처리

## 기술 스택
- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **JSON 파일 기반 데이터 저장** (`data/saved-profiles.json`)

## UI/UX 요구사항
- 모바일 우선 반응형 디자인 (앱 화면 대응)
- 다크 그라디언트 테마 (인디고 → 슬레이트)
- 최소 터치 타겟 44px
- iOS safe area 대응
- 부드러운 전환 효과
